## Oil Skin - Repositorio Oficial

Deployment del proyecyo:
1. Crear un virtual environment.
2. Crear las variables de entorno de la base de datos con los siguientes valores:
- MYSQL_HOST=
- MYSQL_USER=
- MYSQL_PASSWORD=
- MYSQL_DB=
3. Instalar las dependencias del proyecto usando pip con el siguiente comando:
- pip install -r requirements.txt
4. Ejecutar el archivo app.py usando python o el debugger de su preferencia.

Tambien puede descargar el codigo fuente desde github releases.
